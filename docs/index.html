getCognos

getCognos - Static function to get the Cognos Object. You can have only 1 Cognos object in your application at any time.

getCognos(url: String, debug: Boolean): Promise
Parameters
url (String = false) The URL of your Cognos installation. If empty, this function becomes static and a Promise for the current jCognos object is returned.
debug (Boolean = false) If true, starts debugging into the console
Returns
Promise: a promise that will return the jCognos object
